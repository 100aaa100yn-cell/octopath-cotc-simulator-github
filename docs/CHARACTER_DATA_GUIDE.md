# キャラクター詳細データ入力ガイド

## 基本手順

1. 画面の「詳細CSV雛形」をダウンロードします。
2. 既存キャラクターを更新する場合は `id` と `name` を必ず残します。
3. 分からない項目は空欄のままにします。空欄は既存値を上書きしません。
4. 複数値はセミコロン区切りで入力します。例: `技A;技B`。
5. 出典URLと確認日を `sourceUrl` / `sourceCheckedAt` に記録します。

## 完成判定

能力値12項目と、サポート・バトル・必殺技の3分類が揃うと「詳細入力済み」です。EX技はキャラクターによって存在しないため必須ではありません。

## データ品質

- `incomplete`: 名前など最低限のみ
- `provisional`: 出典確認済みだが再確認余地あり
- `verified`: 複数確認またはゲーム内確認済み
- `simulator`: 動作確認用の架空データ
