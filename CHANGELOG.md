# v3.1.0

- マイ旅団でアビリティ5枠・武器・頭/体/腕防具を設定
- ターン戦闘へロードアウトと装備補正を反映
- 推定最短撃破ターン探索を追加
- 旅団JSONをv3.1形式へ更新

# v3.0.0

- データ品質ダッシュボードを追加
- 全276キャラクターにバトルアビリティ3枠・必殺技・EXアビリティの選択枠を保証
- 未登録能力は捏造せず「未登録」仮枠として明示
- ターン戦闘の能力選択肢へカテゴリと品質状態を表示

# v2.9.0

- 勝ち筋探索（戦略候補の自動生成）を追加
- 最短撃破・最大火力・SP効率・総合の4目標を追加
- Boost集中、早期ブレイク、SP温存、交代循環、火力前倒し、3ターン周期の候補を生成
- 候補ごとの推定火力・盾削り・SP・Boost・交代回数を表示
- 候補を元プランを壊さず新しい攻略プランとして適用
- 候補JSON出力を追加
