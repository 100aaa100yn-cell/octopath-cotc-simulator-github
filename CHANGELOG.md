# Changelog

## [2.2.0] - 2026-07-21

### Added
- バフ・デバフの効果量、重複上限、残りターン管理
- 物防・属防DOWN、必ず会心、ダメージ上限UP対応
- 効果セットのブラウザ保存とJSON入出力
- 登録効果を本格ダメージ計算へ自動反映


## [2.1.0] - 2026-07-21

### Added
- 武器・頭・胴・腕・アクセサリー2枠の装備管理
- 5枠のソウル補正編集
- ダメージ上限・物理/属性ダメージ・会心補正
- 装備テンプレート保存・適用・削除
- 装備JSONバックアップ・復元とブラウザ自動保存

### Changed
- 旧armor枠をbody枠へ自動移行


## [2.0.0] - 2026-07-21

### Added
- マイ旅団による所持・未所持管理
- レベル、覚醒、必殺技レベル、EX習得状況の登録
- 編成対象の個別切替と自動編成連動
- 旅団JSONのバックアップ・復元
- 所持数、編成対象数、覚醒IV、EX習得数の集計


## [1.9.0] - 2026-07-21

### Added
- 敵の武器・属性弱点を反映する自動編成スコア
- 弱点連撃、シールド削り、火力、速度、支援効果の評価内訳
- パーティ全体の弱点カバー率と未対応弱点表示
- 同じ弱点担当への偏りを抑える多様性ボーナス
- 敵別候補ランキング

### Changed
- おすすめ編成を固定スコア順から敵別最適化へ変更


## [1.8.0] - 2026-07-21

### Added
- HP割合による敵フェーズ自動切替
- フェーズ別の弱点・シールド・防御・倍率・行動パターン反映
- 開始フェーズ選択と自動切替ON/OFF
- 戦闘ログへのフェーズ移行イベント・現在弱点表示
- 到達フェーズ数の戦闘サマリー
- 3フェーズ訓練敵のサンプルデータ


## [1.7.0] - 2026-07-21

### Added
- 敵詳細データCSVの取込・雛形・未入力レポート
- HP、レベル、防御、シールド回復、弱点、フェーズ、行動パターン、出典の管理
- 敵データ完成度の集計

### Changed
- 敵追加フォームを詳細項目に対応
- 敵データ検証を強化


## [1.6.0] - 2026-07-21

### Added
- キャラクター詳細CSVテンプレート
- データ完成度と不足項目集計
- 未入力キャラクターCSVレポート
- 出典URL・確認日の管理

### Fixed
- CSVの空欄が既存データを消す問題
- HP・防御・会心・配列項目の型変換不足


## [1.5.2] - 2026-07-21

### Fixed
- GitHub Pagesで古いJavaScriptが残る問題を防ぐキャッシュバスターを追加
- JSON取得をモジュールURL基準に変更し、サブパス公開でも安定化
- データ取得時にブラウザキャッシュを使用しないよう変更
- 画面右下に実行中のバージョンを表示


## [1.5.1] - 2026-07-21

### Fixed
- キャラクター選択画面から欠落していた技CSV取込・技CSV雛形ボタンを復元
- GitHub Pages上の初期化エラーを誤ってLive Serverエラーと表示する問題を修正
- 起動失敗時に実際のエラー内容を表示


## [1.5.0] - 2026-07-21

### Added
- ベースランク（★5・★4・★3）の表示・検索・集計
- 追加キャラクター 185 人
- キャラクターCSVのbaseRank・job・acquisition対応
- 未検証キャラクターをincompleteとして明示


## [1.4.0] - 2026-07-21

### Added
- キャラクター／アビリティのデータカタログ集計
- アビリティCSV一括取込
- アビリティCSVテンプレート
- 技名・タグを含む横断検索
- 技分類フィルターと技登録済みフィルター
- データ品質・シリーズ・タグ検証
- キャラクター別の技登録件数表示

### Fixed
- Character CSV取込時のDataManager互換メソッド不足


## v1.3.0

- 武器・防具・アクセサリー2枠を追加
- キャラクター別装備セットをプリセット・共有URLへ保存
- 装備可能な武器種を自動制限
- 装備補正を本格ダメージ計算へ反映
- 装備データのJSON編集・検証に対応
- 初期装備16件を追加（シミュレーター用暫定値）
